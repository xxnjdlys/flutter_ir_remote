# flutter_ir_remote

A Flutter application which allows you control Tv/Box By IR.

## Getting Started

- install dependencies by
```
flutter pug get
```

- run project by
```
flutter run
```

### Supported Device
- Xiaomi TV/Box (小米电视/盒子)
- Tmall Box(天猫魔盒)
- Hisense TV(海信电视)
- Skyworth TV(创维酷开电视)
